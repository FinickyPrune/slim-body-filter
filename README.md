# SlimBody App
## Brielf Description
Pet app developed to get familiar with:
  - Metal Framework
  - Metal Shading Language
  - CoreImage Framework

## Flow
1) Select photo from Photos
2) Select filter from the collection (Blur, Brightness or SlimBody)
3) Control sensitivity
4) Save photo back to Photos
